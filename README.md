# Face Detection using OpenCV

This project detects faces in real-time using a webcam and OpenCV in Python.

## Technologies Used
- Python
- OpenCV

## How to Run
1. Install OpenCV:
pip install opencv-python

2. Run the program:
python face-reco.py


3. Press `ESC` key to exit the webcam window.

## Files Included
- `face-reco.py` - Python script for face detection
- `haarcascade_frontalface_default.xml` - Pre-trained Haar Cascade model for face detection
